Installation
-------------------------------------------------------------------------------------------------

1. Replace the images with your own under /assets/images/ folder.
2. Edit index.html file and modify the content of your own.
3. Upload index.html file and assets folder inside a same folder on your hosting server.
4. Open the path in browser to see the live site.
